# Ситстема генерации CSV-данных

Генерация и анализ CSV-данные с помощью разных Docker-контейнеров

## Установка

```bash
git clone https://github.com/yonglore/HW3.git
cd HW3
```

## Команды

```bash
./run.sh build_generator
./run.sh run_generator
./run.sh create_local_data

./run.sh build_reporter
./run.sh run_reporter

./run.sh structure
./run.sh clear_data
./run.sh inside_generator
./run.sh inside_reporter
```
